Software Design Studio (CS 126)

Learned best practices for writing code including clear documentation, easy to understand function/variable names, and the use of whitespace to block code together
Heavily used elements of Object Oriented Programming including interfaces, abstract classes, inheritance, abstraction, and closed object fields
Used Java to create a text based adventure game using JSON for storage, a Crazy 8s card game implementation, and a TicTacToe winner checker
Used C++ to create a Sudoku solver, a hand written digit classifier, and a music visualizer with the Cinder framework for the final project

Discrete Structures (CS 173)

Learned principles of number theory and logic with proofs, modular arithmetic, sets, relations/functions, and induction
Explored theoretical computer science principles with graphs, trees, and recursive definitions
Examined algorithms like merge sort and Karatsuba using Big-O notation, unrolling, recursion trees, and state diagrams

Data Structures and Algorithms (CS 225)

Learned fundamental data structures including stacks, queues, linked lists, BSTs, hash tables, and disjoint sets
Implemented algorithms to manipulate and create images based on traversals like DFS and BFS
Created a CSV parser to create and manipulate a graph of the Texas road network to find the shortest and landmark path for different points

Computer Architecture (CS 233)

Created circuits like adders, multiplexers, flip flops, and registers using Verilog
Implemented a pipelined CPU based on the MIPS ISA with instruction parsing, memory manipulation, conditionals, branches, and jumps
Translated C programs like a dominoe solver into assembly
Converted programs into vectorized forms using SIMD instructions in C

Systems Programming (CS 241)

Built a shell to execute commands, track history, load and run script files, and pass I/O between programs
Created implementations of malloc, calloc, realloc, and free that were faster than glibc and placed top 10 in classwide competition
Implemented Make build tool that built dependencies in parallel using a thread pool based on pthreads
Created a fast FTP client and server using epoll to support multiple requests and interruptions

Numerical Methods (CS 357)

Learned about floating point computation, storage of floats and doubles in memory, and different properties of these systems
Implemented optimization algorithms for N-Dimensional equations using Newton's method, the golden section method, and the bisection method
Ran Principal Component Analysis on a breast cancer data set to predict benign and malignant tumors
Used linear algebra with Markov chains, linear regressions, and state modeling

Probability & Statistics for Computer Science (CS 361)

Used Numpy, Pandas, and Matplotlib to visualize and find relations in data
Created large simulations to predict probabilities for different events
Implemented ADAM optimizer and tested effectiveness with different parameters and data
Used different probability distributions to make accurate predictions about events

Fundamental Mathematics (MATH 347)

Proved relations between numbers using remainders, congruence, greatest common divisors, and the Euclidean Algorithm
Learned principles of Real Analysis including well ordering, the Completeness Axiom, and the infimum and supremum

Abstract Linear Algebra (MATH 416)

Proved theorems on abstract vector spaces including real numbers, complex numbers, polynomials, and matrices

Differential Equations (MATH 441)

Used Picard Iterates to prove Existence-Uniqueness Theorem

Statistical Analysis (STAT 200)

Learned proper experimental design, confounders, and double blind procedures
